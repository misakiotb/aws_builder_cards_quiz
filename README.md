# AWS BuilderCards Quiz

AWSのアーキテクチャ学習カードゲーム「AWS BuilderCards」を題材にした、学習用クイズアプリです。
現在は初期段階として、"TCO credit" 種別のカードをランダムに5枚表示する機能を実装しています。

## 機能
- **カード表示**: "TCO credit" カードの中からランダムに5枚を選出して表示します。
- **ボードゲーム風デザイン**: 実際のカードゲームのようなリッチなデザインを採用しています。
- **レスポンシブ対応**: PCおよびモバイル端末での表示に対応しています。

## 使用技術
- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)

## 実行方法
本アプリは静的なWebサイトとして動作します。
`index.html` をブラウザで開くだけで動作確認が可能です。

## データソース
AWS BuilderCardsのカードリストデータを使用しています。
※現在は `script.js` 内にデータを埋め込んでいます。
