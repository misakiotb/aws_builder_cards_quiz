# AWS BuilderCards Quiz

AWSのアーキテクチャ学習カードゲーム「AWS BuilderCards」を題材にした、学習用クイズアプリです。
現在は初期段階として、"TCO credit"、"AWSome credit" 種別のカードをランダムに5枚表示する機能を実装しています。

## 機能
- **カード表示**: "TCO credit"、"AWSome credit" カードの中からランダムに5枚を選出して表示します。
- **ボードゲーム風デザイン**: 実際のカードゲームのようなリッチなデザインを採用しています。
- **レスポンシブ対応**: PCおよびモバイル端末での表示に対応しています。

## 使用技術
- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)

## 実行方法
本アプリは静的なWebサイトとして動作します。
`index.html` をブラウザで開くだけで動作確認が可能です。

## データソース
AWS BuilderCards v1 のカードリストデータを使用しています。
※現在は `data.js` 内にデータを埋め込んでいます。

## 構築にあたるAIの利用
本アプリは、AIコーディングツール Google Antigravity を利用した Vibe Coding で構築しました。
サービスの解説文は Claude で作成しており、順次説明の正しさを検証中です。
